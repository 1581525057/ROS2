## 本项目用到的消息类型

### geometry_msgs/msg/Twist

控制机器人速度的标准消息，`turtlesim` 用这个来接收运动指令。

```cpp
// C++ 使用方式
#include "geometry_msgs/msg/twist.hpp"

auto msg = geometry_msgs::msg::Twist();
msg.linear.x = 1.0;    // 线速度 x（前进）
msg.angular.z = 0.5;   // 角速度 z（转弯）
publisher_->publish(msg);
```

字段说明：
- `linear.x` — 前进/后退速度
- `angular.z` — 左/右旋转速度（平面机器人只用 z 轴）
- 两者同时非零 → 圆周运动

---

### example_interfaces/msg/String

最简单的字符串消息，只有一个 `data` 字段。

```python
# Python 使用方式
from example_interfaces.msg import String

msg = String()
msg.data = "要发送的文本内容"
self.publisher_.publish(msg)
```
